## P1 + P2 研究結論

### 已回答的問題

1. **台積電為何不被選入？** → `max_same_industry=2` 把它擠掉。2330 排名通常在 5-15，但同產業的高排名股已佔滿 2 個名額
2. **hold_buffer=3 是否造成滯後？** → 否。hold_buffer 2 和 3 在 tn=8 下結果完全一致
3. **top_n=8 是否太集中？** → 否。top_n=10/12 反而稀釋 alpha，8 是最佳
4. **institutional_flow 是否有預測力？** → 否。rank IC 全期 -0.053（2022: -0.054, 2023: -0.136, 2024-H1: +0.105, 2024-H2: -0.041）。移除後 vol/beta 不變，改善純粹來自排除噪音因子
5. **caution exposure 應該調高嗎？** → 否。78% 回測期為 caution/risk_off，調高等於在 78% 的 in-sample 加槓桿。Vol 上升 7%、beta 上升 3.4%，不是選股改善而是市場曝險增加
6. **trend_quality 需要改進嗎？** → ROI 過低。唯一二元子因子（structure）僅佔總分 3%，改進效果有限
7. **持股數量可以縮減嗎？** → 否。top_n=3 的 3Y Sharpe 掉到 1.05（-43%），vol 升到 31%。top_n=5 更差（Alpha 轉負）。**top_n=8 是最佳**
8. **產業限制可以拿掉嗎？** → 否。無限制時 6M Alpha -3.67%，3Y Sharpe 1.57。全押電子在產業回檔時無緩衝。**max_same_industry=3 是最佳**
9. **FinMind 資料準確嗎？** → 是。比對台積電 2024-12 月 22 天 OHLCV，與 TWSE 官網零誤差
10. **Revenue 覆蓋率是問題嗎？** → 否。實際 ~99%，僅 6805/1802 偶發缺失。金融股在 FinMind 有完整月營收

### 關鍵判斷

- `max_same_industry` 2→3：**結構性改進**，已落地 ✅（P1，Codex + Claude 驗證通過）
- `institutional_flow` 10→0%：**噪音因子移除**，已落地 ✅（P2，Codex + Claude 雙重驗證通過）
- `caution` exposure 調整：**overfit 風險太高**，不落地 ❌（vol +7%, beta +3.4%, 78% in-sample 偏差）
- **下一步：paper trading 累積 6 個月 out-of-sample 數據**

## P3 策略擴展研究結論

### 已測試的方向

| 方向 | 結果 | 判定 |
|------|------|------|
| P3.1 Revenue 覆蓋率 | 實際 ~99%，非問題 | ✅ 結案 |

| P3.4 四因子（+quality） | 6M Sharpe 1.08→0.69，3Y 1.85→1.79 | ❌ 不落地 |

### 核心發現

**動能策略有一個根本特性：alpha 來自追強勢股。**

任何試圖壓制強勢股的做法都會傷害績效：
- 波動率加權 → 高動能股通常波動大 → 被降權 → alpha 消失
- 品質因子 → 穩定公司通常動能弱 → 稀釋動能權重 → alpha 被稀釋

**目前三因子（PM 55% + TQ 20% + RM 25%）已是最佳組合，不需要新增因子。**

## 研究態度

- P1 找到結構性問題（max_same_industry），P2 找到噪音因子（institutional_flow），P3 確認三因子已是最佳
- Exposure 調整是典型的 in-sample overfit 陷阱：78% caution 期 + 最終牛市 = 調高必然好看
- 落地策略：一次改一項，每項都要 Codex 複核 + out-of-sample 驗證
- 區分「選股改善」和「加槓桿」：看 vol 和 beta 是否變化
- P1+P2 已驗證此方法論有效：IF 移除 vol/beta 不變 → 真選股改善；caution 調高 vol/beta 上升 → 加槓桿假象
- **P3 的結論：不是所有「理論上好」的改進都值得做。波動率加權和品質因子在文獻中有效，但在動能策略中反而傷害績效。**

## Codex 對 P3 結論的補充（2026-03-31）

### 我同意的部分

- 波動率加權拖累目前策略，邏輯上是成立的。  
  這套策略的 alpha 主要來自強勢動能股，而這些股票常伴隨較高波動；用 `1 / vol` 去降權，本質上是在削弱 alpha 來源。

- 品質因子加入後稀釋動能，現有 artifact 也支持這個方向。

### 我不同意講太滿的部分

- 在目前已測的 P3 兩個方向中，三因子 baseline 較佳

### 研究上更合理的下一句話

- `vol_weighted`：可視為暫時淘汰
